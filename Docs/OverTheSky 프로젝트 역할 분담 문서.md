## 팀장(이성규) - 메인 시스템 / 플레이어

### 주말 사전 작업
- 기본 시스템 및 베이스 클래스 제작
- 기본 이동/점프
- 바닥 감지 (SphereCast)
- 애니메이션 연동
- Rigidbody 세팅

### 주중 작업
- 경사 처리 (ProjectOnPlane)
- 천장/벽 감지
- 3인칭 카메라
- 기믹 베이스 클래스 제공
- 팀원 기술 지원

### 산출물
- Player.prefab
- MainCamera.prefab
- MovingPlatform.cs (베이스 클래스)
- Hazard.cs (인터페이스)
- TriggerZone.cs (베이스 클래스)
- GameManager를 비롯한 시스템 설계
- 각종 문서 작업물

## (팀원) 허범 - 이동 기믹

### 구현 목록

**1. PingPongPlatform (왕복 발판)**
- A ↔ B 지점 왕복 이동
- 이동 속도/대기 시간 조절

**2. CircularPlatform (원형 발판)**
- 중심점 기준 원형 회전
- 회전 속도/반경 조절

**3. WaypointPlatform (경로 발판)**
- 여러 웨이포인트 순차 이동
- 경로 설정 가능

**4. RotatingObstacle (회전 장애물)**
- 제자리 회전
- 플레이어 충돌 시 넉백

### 산출물
- 기믹 프리팹 4종
- 구현 문서

---

## (팀원) 이인 - 파괴 기믹

### 구현 목록

**1. BreakablePlatform (기본)**
- 밟으면 1.5초 후 부서짐
- 경고 피드백 (색 변경, 흔들림)
- 3초 후 재생성

**2. QuickBreakPlatform (즉시)**
- 밟는 순간 즉시 부서짐
- 2초 후 재생성

**3. FallingObject (낙하 오브젝트)**
- 일정 주기로 위에서 낙하
- 플레이어 충돌 시 넉백
- 땅 닿으면 파괴 후 재생성

### 산출물
- 기믹 프리팹 3종
- 구현 문서

---

## (팀원) 조규민 - 환경 기믹

### 구현 목록

**1. WindZone (바람 구역)**
- 플레이어에게 바람 힘 적용
- 방향/세기 조절
- 시각 피드백 (파티클)

**2. GravityReverseZone (중력 반전)**
- 구역 진입 시 중력 반전
- 구역 탈출 시 정상 복귀

**3. SpeedZone (속도 구역)**
- 이동 속도 변경 (가속/감속)
- 시각 피드백

### 산출물
- 기믹 프리팹 3종
- 구현 문서

---

## (팀원) 김동현 - 기믹(자유), 스폰 시스템/맵

### 구현 목록

**1. CheckpointSystem**
- 체크포인트 도달 시 저장
- 활성화 시각 피드백

**2. RespawnSystem**
- 떨어지면 리스폰
- 마지막 체크포인트로 이동
- 카운트다운 (3, 2, 1...)

**3. UI**
- 현재 높이 표시
- 체크포인트 알림
- 리스폰 카운트다운
- 조작법 안내

**4. 맵 디자인**
- 시작 지점
- 체크포인트 3곳
- 최종 목표 지점
- 난이도 곡선 설계

**기믹(자유)**
- 자유롭게 자신이 생각하는 기믹 및 연출을 구현.

### 산출물
- Checkpoint.prefab
- UI Canvas
- 완성된 맵
- 레벨 디자인 문서
- 기믹
