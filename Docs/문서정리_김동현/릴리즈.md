# 게임 개요
각종 기믹과 함정들을 피하여 골인지점까지 도달하는 게임

# 조작 방법
입력키          동작
W/A/S/D         앞/좌/뒤/우 이동
Space           점프
R               리스폰

# 주요 구현 내용
- 플레이어 이동 및 점프 로직
- 중앙점을 기준으로 공전하는 발판
- 끝점을 기준으로 좌/우 이동하는 발판
- 플레이어가 밟을 시 1.5초 뒤 사라지는 장판
- 체크 포인트 시스템
- 플레이어가 떨어지는 낙하 오브젝트에 닿을 시 날라보냄
- 기본 UI
- 특정 구역의 발판들을 가리는 안개
- 회전하는 기둥의 플레이어를 날리는 기둥
- 트리거에 닿을 시 차가 소환되는 구역

# 실행 방법
1. 첨부된 zip파일 다운로드
2. 압축 해제
3. ```OverTheSky.exe```실행
※ 별도의 설치 과정은 필요하지 않음

# 주의 사항
- 게임 플레이시 화가 날 수 있음
