# Over The Sky

이 프로젝트는 플랫폼 장르의 게임이다.
달리기와 점프로 발판을 밟고 장애물을 피하며 가장 높은 곳에 있는 목표지점에 도달하면 된다.

- 장르 : 플랫폼
- 플레이타임 : 약 1시간
- 개발 환경 : Unity 2022.3.62f2
- 프로젝트 기간 : 2026.01.28 ~ 2026.02.03
- 팀원
- 이성규
- 김동현
- 조규민
- 이인
- 허범
